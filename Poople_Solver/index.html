How this solver works
  • Enter any four-letter word and select Solve.
  • Each row differs from the previous row by one highlighted letter.
  • Use Hint mode for ranked next-word choices without revealing the full solution.
  • Select Not accepted? to exclude a word and find another route.
  • The dictionary is extracted from Poople’s current public game code.
